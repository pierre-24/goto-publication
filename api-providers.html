RSC¶

class goto_publication.providers.RSC¶

Royal society of Chemistry

No trace of an API, but a painful two-request process to get any result: the actual result page takes a long (mostly base64) payload as POST input, but this payload cannot be forged in advance (it seems to contain, at least, some information on the system which generates it).

get_doi(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

Requires 2 requests. An user-agent is mandatory.

get_journals(**kwargs: dict) → List[goto_publication.journal.Journal]¶

Based on the list found at https://pubs.rsc.org/en/Journals. Need an user-agent.

get_url(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

Forged based on the search form: requires no request

ScienceDirect (API)¶

class goto_publication.providers.ScienceDirectAPI(api_key: str = '', concepts: List[Any] = None)¶

Science Direct (Elsevier) API.

Getting the DOI requires a valid API key (Get one at https://dev.elsevier.com/index.html).

Warning

Note that for this kind of usage, the person who uses the Elsevier API needs to be a member of an organization that subscribed to an Elsevier product (see https://dev.elsevier.com/policy.html, section “Federated Search”).

get_doi(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

Use the API in one single request.

get_journals(**kwargs: dict) → List[goto_publication.journal.Journal]¶

Use the API. Assumption: by looking for “volume 1, page 1”, one gets every first page of every journal.

Springer¶

class goto_publication.providers.Springer(concepts: List[Any] = None)¶

Springer have a messed up notation for its article (mixing between pages and article number) and their API (https://dev.springernature.com/adding-constraints) does not provide a page or article number search (and does not give the article number anyway, except e-location in the jats output, which is sloooooow).

Thus, it is impossible to get the exact URL or DOI without any further information.

get_journals(**kwargs: dict) → List[goto_publication.journal.Journal]¶

Get the journal based on their search form. Use the CSV output (that was unexpected!).

get_url(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

Go to TOC of the volume, find your way into that ;)

Wiley¶

class goto_publication.providers.Wiley(concepts: List[Any] = None)¶

Wiley.

Perform the request on their search page API, since there is no other correct API available. Sorry about that, open to any suggestion.

get_doi(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

The DOI is actually found in the URL, so requires one single request as well.

get_journals(**kwargs: dict) → List[goto_publication.journal.Journal]¶

Get the list from https://onlinelibrary.wiley.com/action/showPublications. Since the list is actually long (and the request takes longer if the pages are large), fetch the results 50 by 50 (I didn’t try to optimize that).

get_url(journal_identifier: Any, volume: [<class 'str'>, <class 'int'>], page: str, **kwargs: dict) → str¶

Require a single request to get the url (which contains the DOI).
